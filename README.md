# ID to ISBN

A command line program that converts ids of the form "^\d{12}$" to their ISBN equivalents. Has built in tests, which can be run with the test command (refer below).

## Execute

Compile

```
javac IdToISBN.java
```

Run

```
java IdToISBN run 978155192370 978140007917
```

Test

```
java IdToISBN test
```
